# 说明
请将此文件保存为 config.json，并与 sing-box.exe 放在同一目录下。

# 封装服务
1. 下载 NSSM。
2. 在 CMD 中执行：nssm install sing-box。
3. 在弹出的窗口中： 
   Path: 选择 sing-box.exe。
   Startup directory: 填入所在的文件夹。
   Arguments: 填入 run。
4. 在“服务”管理工具中，将 sing-box 服务的登录账户设置为 “Local System” 或你的 管理员账户，并确保它在 VPN 启动之后或同时运行。

# 程序伪装
可以直接把 sing-box.exe 重命名为 svchost.exe 或 msedge.exe 后再运行。这样在任务管理器和 VPN 审计日志里，它看起来就是一个标准的系统组件。

# 小贴士
在 Win2016 上，应用层 VPN 有时会修改 Windows Firewall 规则。请确保你在防火墙中放行了 “指定的程序”（即更名后的 msedge.exe）的所有入站和出站权限，而不仅仅是端口。
